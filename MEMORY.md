## 执行原则：原生优先，脚本兜底

> 来源：林总指令 | 2026-06-19

### 优先级序列
1. **OpenClaw 原生内置工具**：文件读写、exec、browser 等
2. **ClawHub 官方验证技能**：优先安装使用官方成熟技能
3. **自定义脚本**：仅在原生能力和官方技能完全无法覆盖时，作为最后兜底

### 禁用规则
- ❌ 能用原生工具解决却写自定义脚本
- ❌ 能用官方技能解决却写复杂脚本链
- ❌ 留存无人维护的临时脚本（临时脚本用完立即清理）

### Cron Job 硬性约束
> 来源：根因复盘 | 2026-06-20 | 强制执行

**核心原则：禁止 agent 自由构造 shell/PowerShell/git 命令。**

| 约束类型 | 具体要求 |
|----------|----------|
| ❌ 绝对禁止 | `exec` 运行 PowerShell/git/CMD 命令 |
| ❌ 绝对禁止 | `$OPENCLAW_HOME`、`$env:XXX`、`$(Get-Date)` 等变量 |
| ❌ 绝对禁止 | 让 agent 动态生成脚本并执行 |
| ✅ 允许 | `read`/`write`/`edit` 等内置文件工具 |
| ✅ 允许 | Python + 绝对路径脚本 |
